# MiniAds

### Dependencies
* MongoDB 
* Rails 4
* NojeJS
* bower
* bundle


### Setup project
* `git clone git@github.com:JonnatasCabral/MiniAds.git`
* `$ cd MiniAds/`
* `bundle install`
* `bower install`


### Run project
* `rails server`
