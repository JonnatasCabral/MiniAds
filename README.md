# MiniAds

### Dependencies
* MongoDB 
* Rails 4
* NojeJS
* Bower
* Bundle
* Ruby 1.9.3 `gem 'debugger'`  or ruby 2.+ `gem 'pry-byebug'`



### Setup project
* `git clone git@github.com:JonnatasCabral/MiniAds.git`
* `$ cd MiniAds/`
* `bundle install`
* `bower install`


### Run project
* `rails server`
